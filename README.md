# Business Unit Roles Explorer for [XrmToolBox](http://www.xrmtoolbox.com)

![Nuget](https://img.shields.io/nuget/dt/Fic.XTB.FlowExecutionHistory) ![Nuget](https://img.shields.io/nuget/v/Fic.XTB.FlowExecutionHistory)


Business Unit Roles Explorer is an XrmToolBox plugin that allows administrators to easily browse, assign, remove, and copy security roles for users across business units in Microsoft Dynamics 365 / Dataverse.

The tool is designed to simplify day-to-day security role management by providing a fast, searchable user list and clear visibility into role assignments per business unit.

## Features

### User Search & Filtering

Quickly find users using a responsive search box with debounce logic, making it easy to work even in large environments with many users.

### View User Roles by Business Unit

Select a user to instantly see all assigned security roles along with their associated business units in a clear grid view.

### Add/Remove Security Roles

Add/remove one or multiple security roles to a selected user using a dedicated dialog, ensuring roles are added correctly through the native Dataverse association mechanism.

### Copy Roles Between Users

Copy all security role assignments from one user to another, streamlining onboarding and role alignment across teams or business units.
